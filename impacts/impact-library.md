# Impact Library

> Reusable, CV-ready bullets grouped by impact type. Keep each bullet concise, owned, and quantified where possible.
>
> **Bullet format:** `Verb + what you did + measurable outcome`
> Example: `Reduced deployment failures by 40% by stabilising pipeline sequencing across 3 regions.`
> Every bullet must pass: (1) starts with a past-tense action verb, (2) contains a number or scale metric, (3) answers "so what?" for a hiring manager.
>
> **Tagging convention:** add inline Dataview fields so CV versions can query by role type.
> Format: `- Bullet text. [tags:: backend, reliability] [company:: YourCompany]`
